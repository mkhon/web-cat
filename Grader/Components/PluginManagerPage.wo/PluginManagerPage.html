Automatic Updates

Automatic updates: Enabled Disabled

This application is capable of downloading and installing new updates to plug-ins that are currently installed. When enabled, Web-CAT will check for newer versions of all installed plug-ins on each restart, and download any available updates at that time. You can also manually download updates on demand using the download icons in the table. Downloaded updates will be applied immediately.

Automatic Installation

Automatic installation: Enabled Disabled

This application is capable of automatically downloading and installing new plug-ins that are available from known providers. When enabled, Web-CAT will check for brand new plug-ins on each restart, and automatically download and install any that are available at that time. You can also manually install new plug-ins available on the web using the download icons in the table. Downloaded plug-ins will be installed immediately.
