Automatic Updates

Automatic updates: Enabled Disabled

This application is capable of downloading and installing new updates to plug-ins that are currently installed. When enabled, Web-CAT will check for newer versions of all installed plug-ins on each restart, and download any available updates at that time. You can also manually download updates on demand using the download icons in the table. Downloaded updates will be applied immediately.
