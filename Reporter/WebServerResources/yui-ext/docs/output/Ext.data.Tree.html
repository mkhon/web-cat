Class Ext.data.Tree

Package:Ext.data
Class:Tree
Extends:Observable
Subclasses:TreePanel
Defined In:Tree.js
Represents a tree data structure and bubbles all the events for its nodes. The nodes in the tree have most standard DOM functionality.
