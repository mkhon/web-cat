Automatic Updates

Automatic updates are disabled on this installation. Automatic updates are only available on servlet-based deployments of this application. Any updates must be downloaded and applied by hand when the application is not executing.

Automatic updates: Enabled Disabled

This application is capable of downloading and installing new updates to subsystems that are currently installed. When enabled, Web-CAT will check for newer versions of all installed subsystems on each restart, and download any available updates at that time. You can also manually download updates on demand using the download icons in the table. All downloaded updates will be applied when the application restarts.
